# 帮助命令
## man 获取帮助信息(获取帮助信息)
1) 基本语法
```bash
man [命令或配置文件] 
```

# 文件目录类
## pwd 显式当前工作目录的绝对路径
```bash
[root@VM-4-16-centos lib]# pwd
/usr/lib
```
## cd 切换目录
```bash
[root@VM-4-16-centos lib]# cd /usr/bin
[root@VM-4-16-centos bin]# 
```
